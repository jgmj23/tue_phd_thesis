# Source Materials

Use this folder for inputs that inform the thesis but are not themselves thesis source files.

Recommended placement:

- `source-materials/university/`: official thesis rules, style guides, templates, and deadlines.
- `source-materials/advisor/`: advisor comments, committee notes, and meeting artifacts.
- `source-materials/prior-writing/`: existing drafts, proposals, papers, reports, and copied text to adapt.
- `source-materials/references/`: exported citation libraries, BibTeX files, RIS files, CSL files, and reading lists.
- `source-materials/figures/`: existing figure assets received from elsewhere.
- `source-materials/papers/`: local paper PDFs. This folder is ignored by Git by default.
- `source-materials/templates/`: external thesis templates or example projects. Third-party templates should normally stay local-only.

When files are added here, create or update issues in `.agents/issues/` that convert them into thesis work.
